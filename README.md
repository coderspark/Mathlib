fibonacci(n) will run a fibonacci script. You can also do a second paramater of 1 = show steps and 0(defualt) will just return the value

sqrt(n) will calculate the square root

pyth_thereom(a, b) will calculate the hypotenuse with a and b

factorial(n) just calculates n! if n=5 then it will return 1 * 2 * 3 * 4 * 5

sin(n), cos(n), and tan(n)(ignore terms and decimal_places) calculate the sine, cosine, and tangent of N

sum(n) is if n = 5 then it will return 1 + 2 + 3 + 4 + 5

calculate_e() runs a calculation of eulers number e

log and ln will do logrithms with chosen base and ln is log base e
